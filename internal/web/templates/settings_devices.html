UB-as-SPC Device Sync Server

UltraBridge's built-in Supernote Private Cloud server that devices sync against directly. Changes to any field here require an application restart to take effect.

Reverse proxy & hostname: this server listens on its own port (:8089 by default), separate from the main UltraBridge app on :8443 (web UI, CalDAV, Boox WebDAV, MCP). Behind a reverse proxy it needs a dedicated hostname (e.g. sn.example.com) whose entry routes to :8089 and forwards the WebSocket upgrade for /socket.io/. Use a different hostname for the main app — one hostname can't serve both, since the proxy routes by Host header to a single backend port. Point your Supernote at the :8089 hostname.

When enabled, UltraBridge runs the Supernote Private Cloud server that devices connect to (binds the listen address below). Leave it off if you don't sync a Supernote device. Takes effect after a restart.

ForestNote Device Sync

UltraBridge's native device-sync server for ForestNote (and future UB-native clients). Exposes /sync/v1 on the main app port, behind the same authentication as the web UI. Synced notes flow into the OCR / search / RAG pipeline like Supernote and Boox notes. Changes here require an application restart to take effect.

When off, /sync/v1 is not served (default off).

Max ops returned per sync response. Default 500.

Boox Database Maintenance

{{if not .BooxActive}}

Add a Boox source in Settings to enable maintenance tools.

{{else}}

Walk the Boox notes directory and enqueue any .note or .pdf files that aren't tracked in boox_notes. Recovers files that landed on disk via something other than the WebDAV upload hook (rsync, manual copy, pre-fix uploads). Already-tracked files are left alone.

Update boox_notes.created_at to match the YYYYMMDD prefix in filenames where the stored date doesn't match. Files without a date prefix are left alone.

Permanently delete files whose name matches Notebook- followed by a digit (Boox firmware default-named files, e.g. Notebook-1.note, Notebook-12-46_03.note). Removes the boox_notes row, the .note file, the .versions archive, and the rendered cache. Files like Notebook-please transcribe me.note are excluded.

{{end}}
