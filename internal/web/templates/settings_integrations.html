Connection

Two transports are available. Create a bearer token in the Tokens section below and paste it into your client config.

{{.MCPHTTPURL}}

Use this for network-accessible clients. Send the bearer token in an Authorization: Bearer <token> header.

{{.MCPStdioCommand}}

The MCP client spawns this per session. Requires docker on the client host, or an ssh wrapper for remote clients.

{
  "mcpServers": {
    "ultrabridge": {
      "command": "docker",
      "args": ["exec", "-i", "ub-mcp", "ub-mcp"]
    }
  }
}
{
  "mcpServers": {
    "ultrabridge": {
      "url": "{{.MCPHTTPURL}}",
      "headers": { "Authorization": "Bearer YOUR_TOKEN_HERE" }
    }
  }
}
